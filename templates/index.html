Local Ollama Configuration
Local Operation Requirements: This application runs completely locally without any network connections. Requires local Ollama and Gemma 3n model installation.
API Address: http://localhost:11434
Model: Gemma 3 4B
Installation Steps:
1. Download and install Ollama: curl -fsSL https://ollama.ai/install.sh | sh
2. Download Gemma 3 4B model: ollama pull gemma3:4b
3. Start Ollama service: ollama serve
